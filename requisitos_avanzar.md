# Assets Requeridos para "La Ruleta Tragona 3000"

Para avanzar con las historias de usuario HU3, HU4 y HU5, se requieren los siguientes recursos. Favor de colocarlos en las rutas especificadas dentro de `public/`.

## 🎨 Assets 3D (Para HU3 y HU4)
*Ruta sugerida: `public/models/`*

1.  **Modelo de la Ruleta (`ruleta.glb` o `.gltf`):** Un cilindro o disco 3D decorado como plato gigante o tapa de olla. Debe tener divisiones claras para las texturas de los nombres de comida/bebida.
2.  **Modelo del Puntero (`flecha.glb` o `.gltf`):** Un tenedor, cuchillo o mano 3D que servirá como indicador del resultado.

## 🖼️ Assets 2D y Texturas (Para UI y HU3)
*Ruta sugerida: `public/images/`*

1.  **Sprites de Comida para Confeti (`pizza.png`, `burger.png`, `soda.png`):** Imágenes pequeñas con fondo transparente para el efecto de victoria.
2.  **Iconos de Carga (`babeando.png`, `hambre.png`):** Caras divertidas para la animación de "Leyendo jeroglíficos".
3.  **Textura de Mantel (`mantel.jpg`):** Patrón de cuadros de picnic para el fondo del escenario 3D.

## 🔊 Assets de Audio (Para HU4 y HU5)
*Ruta sugerida: `public/audio/`*

1.  **Música de Fondo (`elevator_jazz.mp3`):** Música tonta tipo Bossa Nova para la espera.
2.  **SFX Giro (`clack.mp3`):** Sonido corto de la ruleta girando.
3.  **SFX Victoria (`ta_da.mp3`):** Sonido triunfal de celebración.
4.  **SFX Decepción (`womp_womp.mp3`):** Sonido de trombón triste para opciones "saludables".

---
*Nota: Estos assets son necesarios para que la experiencia visual y auditiva sea completa según los criterios de aceptación.*
